Model Description

The web search (web_search) is currently being updated. We do not recommend using this functionality in the near term. This documentation is outdated; please follow subsequent content updates.
  • Kimi K3 is Kimi’s flagship model for long-horizon coding and end-to-end knowledge work, with a 1M-token context window and industry-leading intelligence. See Introducing Kimi K3.
  • Always reasons and supports configuring its reasoning level with the top-level reasoning_effort field (currently max only). See Thinking Effort.
  • Supports automatic context caching, ToolCalls, JSON Mode, structured output (response_format / JSON Schema), Partial Mode, internet search, and more
  • New API capabilities in K3: tool choice constraints (tool_choice) and dynamically loaded tools — see Kimi K3 API Tool Calling Best Practices for combined usage
